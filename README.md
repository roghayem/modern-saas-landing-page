# 🚀 Modern SaaS Landing Page

A modern, responsive, and visually appealing SaaS Landing Page built using HTML, CSS, and JavaScript. This project showcases a clean and professional startup website with a modern user interface, smooth interactions, and a fully responsive design across all devices.

🚀 Modern and clean SaaS landing page design

## 🚀 How to Run Locally

Clone the repository:

```bash
git clone https://github.com/roghayem/modern-saas-landing-page.git
```

Open the project folder:

```bash
cd modern-saas-landing-page
```

Open `index.html` in your browser.
